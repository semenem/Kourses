На сегодняшний момент на рынке ПО представлен широкий ассортимент систем автоматизированного проектирования для моделирования устройств СВЧ, которые отвечают самым современным требованиям, имеют в своем арсенале большой выбор инструментов, позволяющих решать сложные задачи по моделированию электромагнитных полей. К их числу относится программный продукт компании Altair Engineering FEKO, которая в России успела зарекомендовать себя как точный инструмент для расчета антенн и 3D моделирования электромагнитных полей. Первая версия FEKO была создана в 1991 году на базе научных исследований при Университете Штутгарт, Германия. С тех пор, возможности программного обеспечения FEKO постоянно увеличиваются за счет добавления новых численных методов. Графический пользовательский интерфейс системы с момента появления программы претерпел ряд значительных изменений, по мере расширения функциональных возможностей автоматизированной системы он постепенно усложняется. Основу FEKO составляют программно реализованные численные методы моделирования электромагнитных полей, базовый из которых, метод моментов (МоМ).На сегодняшний момент на рынке ПО представлен широкий ассортимент систем автоматизированного проектирования для моделирования устройств СВЧ, которые отвечают самым современным требованиям, имеют в своем арсенале большой выбор инструментов, позволяющих решать сложные задачи по моделированию электромагнитных полей.
